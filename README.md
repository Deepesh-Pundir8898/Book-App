# Book App

## Introduction

This is a Book Search App that allows users to browse and search for books using the provided API. Users can switch between light and dark modes, sign up or log in, view detailed information about selected books, and filter books by category.

## Features

- **Dark Mode**: Toggle between light and dark themes.
- **User Authentication**: Sign up and log in functionality using localStorage.
- **Book Search and Filter**: Browse and filter books by categories.
- **Book Details**: View detailed information about a selected book.

## Technologies Used

- **HTML/CSS**: For structure and styling.
- **JavaScript**: For functionality and interaction.
- **API Integration**: Using the OMDB API to fetch and display books.
